# 📊 Superstore Sales & Profit Performance Dashboard

## Project Overview

This project presents an interactive Power BI dashboard designed to analyze sales performance, profitability, customer segments, returns, and regional trends using Superstore data.

The dashboard enables business stakeholders to monitor key performance indicators (KPIs), identify profit drivers, evaluate sales trends, and support data-driven decision-making.

---

## Business Problem

Retail organizations need visibility into sales performance, profitability, product categories, customer segments, and return behavior.

This dashboard helps answer key business questions:

- Which categories generate the highest profit?
- Which customer segments contribute the most sales?
- How are sales performing compared to the previous year?
- Which regions and states are driving profitability?
- What impact do returns have on business performance?

## Dashboard Features

### Executive KPI Monitoring
- Total Sales
- Total Profit
- Return Order Percentage
- Year-over-Year Performance Comparison

### Profitability Analysis
- Profit by Category
- Profit by Product
- Profit Root Cause Analysis using Decomposition Tree

### Sales Performance Tracking
- Sales vs Previous Year Trend Analysis
- Regional Performance Analysis
- State-Level Profit Distribution

### Customer Analysis
- Sales by Customer Segment
- Segment Contribution Analysis

### Business Insights Panel
- Automated executive summary highlighting key findings

---

## Key Insights

- Technology category generated the highest profit contribution (36%).
- West region delivered the strongest overall profitability.
- Consumer segment accounted for more than 50% of total sales.
- Return orders decreased by 2.9% year-over-year.
- Profit growth exceeded sales growth during recent periods.
